정보포털.com에 안전하게 접속하기 위해 CA 인증서를 설치해주세요

⚠️ 중요 안내

이 사이트는 자체 서명된 SSL 인증서를 사용하고 있습니다. 안전한 접속을 위해 CA 인증서를 설치한 후 HTTPS로 접속해주세요.

1단계: CA 인증서 다운로드

아래 버튼을 클릭하여 CA 인증서를 다운로드하세요.

📥 CA 인증서 다운로드

2단계: 인증서 설치 방법 선택

아래 두 가지 방법 중 하나를 선택하여 인증서를 설치하세요.

💻 시스템 전체 설치 (권장)

이 방법으로 설치하면 모든 브라우저와 프로그램에서 인증서가 인식됩니다.

Windows:
  1. 다운로드한 bssm_root_ca.crt 파일을 더블클릭
  2. "인증서 설치 마법사"에서 "로컬 컴퓨터" 선택 후 "다음"
  3. "모든 인증서를 다음 저장소에 저장" 선택 후 "찾아보기"
  4. "신뢰할 수 있는 루트 인증 기관" 선택 후 "확인"
  5. "다음" → "마침" 클릭
  6. "예"를 클릭하여 설치 완료
macOS:
  1. 다운로드한 bssm_root_ca.crt 파일을 더블클릭
  2. "키체인 접근"에서 "시스템" 키체인 선택
  3. "항목 추가" 클릭
  4. "신뢰" 섹션에서 "이 인증서를 사용할 때" → "항상 신뢰" 선택
  5. 키체인 접근 종료
Linux (Ubuntu/Debian):
  1. 터미널에서 다음 명령어 실행:
  2. sudo cp bssm_root_ca.crt /usr/local/share/ca-certificates/
  3. sudo update-ca-certificates
  4. 브라우저 재시작
🌐 브라우저별 설치

각 브라우저에서 개별적으로 인증서를 설치합니다.

Chrome / Edge 브라우저:
  1. 브라우저에서 chrome://settings/certificates 접속
  2. "인증 기관" 탭 선택
  3. "가져오기" 버튼 클릭
  4. 다운로드한 bssm_root_ca.crt 파일 선택
  5. "신뢰할 수 있는 루트 인증 기관" 체크 후 "확인"
Firefox 브라우저:
  1. 브라우저에서 about:preferences#privacy 접속
  2. 아래로 스크롤하여 "인증서" → "인증서 보기" 클릭
  3. "인증 기관" 탭 선택
  4. "가져오기" 버튼 클릭
  5. 다운로드한 bssm_root_ca.crt 파일 선택
  6. "이 인증 기관이 웹사이트를 식별하는 것을 신뢰" 체크
  7. "확인" 클릭
Safari 브라우저 (macOS):
  1. 다운로드한 bssm_root_ca.crt 파일을 더블클릭
  2. "키체인 접근"에서 "로그인" 키체인 선택
  3. "항목 추가" 클릭
  4. "신뢰" 섹션에서 "이 인증서를 사용할 때" → "항상 신뢰" 선택
  5. 키체인 접근 종료

3단계: 브라우저 재시작

인증서 설치 후 브라우저를 완전히 종료하고 다시 실행해주세요.

📋 주의사항

  • 시스템 전체 설치를 권장합니다. 한 번 설치하면 모든 프로그램에서 인식됩니다.
  • 인증서 설치 후에도 보안 경고가 나타날 수 있습니다. 이는 정상적인 현상입니다.
  • 경고 메시지에서 "고급" → "안전하지 않은 사이트로 이동"을 선택하세요.
  • 인증서는 한 번만 설치하면 됩니다.
  • 문제가 발생하면 브라우저 캐시를 삭제해보세요.
  • Windows에서 "관리자 권한"이 필요할 수 있습니다.
